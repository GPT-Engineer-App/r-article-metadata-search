# r-article-metadata-search


An R based (not react) journal article metadata search and augmentation database. Collect references from Zotero and augment with information from semantic scholar and other sources

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/r-article-metadata-search.git
cd r-article-metadata-search
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
